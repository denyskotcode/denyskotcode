### 📌 Some Things I've Built

**Work** — Automation & Web Scraping Engineer (2023 – 2026)
Built production scraping systems with browser fingerprint tuning (TLS/JA3), a distributed mobile proxy network across multiple carriers, and a hybrid HTTP/browser strategy for handling both JS-heavy and high-throughput targets.

**Projects**

- **[AI Skill Coach](https://github.com/denyskotcode/skoda-hackathon-datascience-ai-skill-coach)** — 48h hackathon (Škoda Auto × 42 Prague): Streamlit app backed by Azure OpenAI GPT-4.1 that analyzes HR/ERP data, surfaces skill gaps, matches courses from an internal catalog, and gives managers a summary dashboard. Built with a team of 3.

- **[Predictive Maintenance](https://github.com/denyskotcode/skoda_predictive_maintenance_engine_dashboard)** — Binary and multiclass failure prediction for CNC machines using XGBoost and LightGBM on the AI4I 2020 dataset (~10K samples, ~3.4% failure rate). Handled class imbalance, added physics-based features, integrated SHAP for explainability. Interactive Streamlit dashboard with real-time risk scoring.

- **[CO2 Emission Predictor](https://github.com/denyskotcode/vehicle_CO2_emission_analyzer)** — Gradient Boosting model (R²=0.98, MAE ~5 g/km) trained on ~7,400 vehicles. Careful feature engineering to avoid data leakage. Streamlit app with EU energy rating (A–G) and market percentile comparison.

- **[minishell](https://github.com/denyskotcode/minishell)** — POSIX-compliant shell in C: lexer → parser → executor, full pipeline and redirection support, heredoc, environment variable expansion. Zero memory leaks.

- **[Inception](https://github.com/denyskotcode/inception)** — Dockerized infrastructure from scratch: NGINX + TLS, WordPress + PHP-FPM, MariaDB — all on a private bridge network with Docker secrets and named volumes.
